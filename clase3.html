  • El elemento <html> encierra todo el contenido de la página entera
  • <head> es el contenedor que no es visible dentro de la página web
  • <title> es el título que aparece en la pestaña o en la barra de título del navegador
  • <body> es todo el contenido visible a los usuarios web
  • <h1>-<h6> son encabezados, títulos y subtítulos
  • <ul> son listas desordenadas con sus diferentes tipos de viñetas
  • <ol> son listas ordenadas, están marcadas con números
  • <a> es para crear un texto en vínculo <a href="http://templated.co">a desarrollar</a>
